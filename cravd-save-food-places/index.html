Cravd - Save Food Places

08 July 2017 on Pet Projects, React, Firebase

After a while I decided to go public with Cravd [pronounced: krāv, just like craved] I've been using for more than a year.

The basic idea is to be able to save places and get the places ordered by if they're open and distance from current location. Places can be anything from restaurants to historic places, places you want to visit in the future.

Cravd is written in React.js with the help of webpack and without a dedicated server (using Firebase). I mainly wrote this to learn React.js though I really like it now and use it in Tel Aviv and when we visit new cities.

Have fun :)
